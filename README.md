# Wallet backend

Сервис, для отслеживания, расчета, накопления финансов

## Запуск

Нужен путь к YAML: **`CONFIG_PATH`**.

## Конфиг (`cleanenv`: YAML + переменные окружения)

| Блок | Назначение |
|------|------------|
| `env` | `local` / `dev` / `prod` (формат логов). |
| `http_server` | `address` — bind (`0.0.0.0:8080`), таймауты. |
| `auth.jwt.secret` | Секрет HS256 для проверки входящего JWT (задаётся в YAML). |
| `db` | Параметры БД (для миграций/других команд; HTTP-notificator их не использует в main). |

Переменные окружения переопределяют значения из файла там, где в `config.go` указан тег `env:`.
